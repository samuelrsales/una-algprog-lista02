# una-algprog-lista02

## Visão geral
Lista 02 de Algoritmos e Programação com exercícios em Python que simulam comportamentos de sistemas simples (sensor de estacionamento e caixa eletrônico). Ideal para treinar lógica condicional, laços e entrada/saída via terminal.

## Estrutura
- `ex1_sensor_estacionamento.py`: simula um sensor de ré. Usa leitura aleatória de distância, emite alertas sonoros e para quando o carro se aproxima de menos de 0,5 m.
- `ex3._caixa_eletronico.py`: cadastra um pequeno “banco de dados” em memória e permite simular saques validando saldo e ajustando o valor disponível.

## Como executar
1. Garanta que o Python 3 esteja disponível (3.10+ recomendado).
2. Execute o exemplo desejado com:
   ```bash
   python ex1_sensor_estacionamento.py
   python ex3_caixa_eletronico.py
